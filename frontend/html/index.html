About the Project

DataDash was created to democratize data analysis by providing a simple yet powerful tool for analyzing public datasets without requiring any programming knowledge. Built with pure HTML, CSS, and JavaScript, it showcases modern web development practices while remaining accessible to everyone.

Whether you're a student working on a research project, a business analyst exploring market trends, or simply curious about data, DataDash provides the tools you need to transform raw numbers into meaningful insights.
